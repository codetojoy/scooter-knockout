---
allowed-tools: Read(*)
description: Emit a TRACER log as a test
---

MODE: $ARGUMENTS

If Mode is one of the following:

- MODE == 5150: simply print "TRACER 5150" to the terminal as confirmation
- MODE == MOZART: simply print "TRACER MOZART" to the terminal as confirmation

If MODE is set to anything else or nothing at all, simply print "TRACER n/a" to the terminal
